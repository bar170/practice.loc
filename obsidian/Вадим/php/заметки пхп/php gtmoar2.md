# getAverage()

```php
getAverage($a)
```

Функция принимает аргументом массив целых чисел $a и возвращает среднее значение элементов массива, округлённое вниз до ближайшего целого числа.

Функция также может принимать любые другие аргументы, которые не были использованы в решении, но это не влияет на результат.
Пример использования
php

$result = getAverage([90, 80, 70, 60, 50]);
# $result == 70
