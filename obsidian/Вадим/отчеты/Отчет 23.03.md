### 2. Создал базу данных SQLite3

**Команды:**

```bash
cd ~/sites/practice1oc/www
sqlite3 mobile_phones.db
```

Создание таблицы:
sql

CREATE TABLE mobile_phones (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL,
    brand TEXT NOT NULL,
    year INTEGER,
    price REAL,
    description TEXT
);

Использованные SQL-команды
Команда	Описание

Заметки по SQL

    PRIMARY KEY AUTOINCREMENT — автоматическое увеличение id

    TEXT — строковый тип данных

    REAL — числовой тип с плавающей точкой

    В запросах важно указывать правильные типы и кавычки для строк

Полезные команды SQLite3
Команда	Описание
.tables	Показать все таблицы
.schema mobile_phones	Показать структуру таблицы
.exit	Выйти из SQLite3
