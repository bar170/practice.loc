# Get the mean of an array

## Ссылка на задачу
https://www.codewars.com/kata/563e320cee5dddcf77000158

## Описание задачи (перевод)
Конец учебного года, судьбоносный момент — школьный табель. Нужно посчитать средние оценки. Все студенты приходят к вам и просят вычислить их средний балл.

Задача: вернуть среднее значение данного массива, **округленное вниз** до ближайшего целого числа.

Массив никогда не будет пустым.

## Моё решение
```php
function getAverage($a) {
    return floor(array_sum($a) / count($a));
}
