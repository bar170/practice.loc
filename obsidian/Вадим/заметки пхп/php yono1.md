# You only need one - Beginner

## Ссылка на задачу
https://www.codewars.com/kata/57cc975ed542d3148f00015b

## Описание задачи (перевод)
Вам дан массив `a` и значение `x`. Нужно проверить, содержит ли массив это значение.

- `a` может содержать числа или строки
- `x` может быть числом или строкой
- Вернуть `true`, если значение есть в массиве, и `false`, если нет
- Важно: использовать строгое сравнение (===), а не нестрогое (==)

## Моё решение
```php
function solution(array $a, $x): bool {
    return in_array($x, $a, true);
}
