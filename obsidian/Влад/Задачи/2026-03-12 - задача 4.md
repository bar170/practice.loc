
1) Сначала прочитать полностью задание
2) Арендовать домен (выберете любое имя свободное)
3) Арендовать VPS (с ubuntu)

```
я использую https://jino.ru/, но выбирайте любой хостинг. 
На jino скорее смогу подсказать что к чему. Нужно в последствии будет связать vps с вышим доменом, это как-будто проще делается, когда все через одного провайдера делается. Если выберете дргой провайдер, уверен тех поддержка сможет подсказать как связать домен и хостинг
```

3) Связать домен в вашей vps
4) подключиться по ssh к своей машине (vps) 
5) Создать пользователя с правами sudo. под root не работаем. назовем его admin
6) Создать директорию /home/admin/sites/название вашего домена/

```
Например:
/home/admin/sites/wenonezhxx.ru/
или
/home/admin/sites/indi-games.ru/
или
/home/admin/sites/daniil333.ru/
```

7) Перейти в директорию **/home/admin/sites/название вашего домена/**
8) инициировать там git (git init)
9) Добавить удаленный репозиторий с которым мы работаем(git remote add origin git@github.com:bar170/practice.loc.git)
10) сгенерировать ключ ssh или найти, если уже есть (в ~/.ssh)
11) Зайти в свой аккаунт гитхаба и добавить его (так же как вы ключ своих локалок добавляли). Пометьте его как-нибудь (назовите), чтобы понимать какой ключ локалки, а какой сервака  
12) Возвращаемся на ваш сервак - выкачиваем ветку мастер в /**home/admin/sites/название вашего домена**/

```
обычная связка: 
фетч 
чекаут -Б
```


13) Установить и запустить nginx на вашем сервере
14) Настроить хост nginx на /home/admin/sites/название вашего домена<u>/www/</u>, чтобы он там видел файл index.php
15) Открыть в браузере ваш сайт (daniil333.ru) - должны увидеть приветственную страницу
16) Делаем скрины (перекидываем их в папку image в обсидиане), заполняем отчет пушим вашу ветку, делаем черрипик на актуальный мастер, пушим мастер
