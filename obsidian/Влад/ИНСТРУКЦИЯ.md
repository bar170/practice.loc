Функции:
1) Заметка функции должны содержать в заголовке только свое название и круглые скобки. Пример: functionName()
2) Описание должно начинаться с записи функции в блоке php. Пример: 
```php
functionName()
```
3) Если функция принимает значения, они должны быть прописаны в блоке php и иметь вид `$argument`. Пример: 
```php
functionName($x, $y, $z, $anyName)
```
4) Описание функции должно содержать слова "Принимает" и "Возвращает". Функция принимает аргументы, а не число, строку и т.д. Пример: Функция принимает аргументом массив `$x` и возвращает строку `$y`.
5) В описании необходимо прописать в полной мере, что делает функция наиболее кратко и понятно.
6) Функции могут принимать аргументы, которые вы не использовали в работе. Такие аргументы записывать не нужно, т.к. суть knowgle в хранении того, что вы усвоили, но можете забыть позднее.
7) Исходя из п.6 описание должно содержаться запись того, что функция может принимать аргументы, которые вы не использовали, если таковые имеются.
8) После описания функции должен идти блок с примером использования функции.
9) В примере использовать комментарий после записи функции с содержанием результата ее работы. Пример:
```php
$result = functionGetSum(5, 6);
# $result == 11
```
10) В конце файла должны содержаться теги #tag подходящие под содержание заметки. 
11) Проверять в ходе решения задач заметки используемых функции и дополнять их (если, например, использовали новые аргументы этих функций).
12) Заполнять можно по шаблону [[ШАБЛОН()]] (ПКМ на ШАБЛОН() -> создать копию -> переименовать ШАБЛОН() 1).
