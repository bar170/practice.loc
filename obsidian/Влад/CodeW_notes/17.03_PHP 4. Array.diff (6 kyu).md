Implement a function that computes the difference between two lists. The function should remove all occurrences of elements from the first list (`a`) that are present in the second list (`b`). The order of elements in the first list should be preserved in the result.

## Examples

If `a = [1, 2]` and `b = [1]`, the result should be `[2]`.

If `a = [1, 2, 2, 2, 3]` and `b = [2]`, the result should be `[1, 3]`.

Solution:

```php
function arrayDiff($a, $b) { 
  $a = array_values(array_diff($a, $b));
  return $a;
}
```

Сводка:

Функция принимает два массива `$a` и `$b` и должна вернуть массив содержащий элементы из массива `$a` за исключением тех, которые есть в массиве `$b`.

В задаче есть небольшой подвох: при использовании функции `array_diff()` сохраняются ключи элементов. Т.е. если удаляется первый элемент, то в новом массиве первое вхождение будет с ключем 1, а не 0. Чтобы это исправить используется функция `array_values()`, которая принимает массив и пересобирает его с новыми ключами, начиная с ключа 0. Чтобы это понять необходимо было использовать функцию var_dump(), которая показывает информации о принятом аргументе, в нашем случае - массива после array_diff().

Использованы: [[array_diff()]], [[array_values()]]
