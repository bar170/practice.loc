Write a function that takes in a string of one or more words, and returns the same string, but with all words that have five or more letters reversed (just like the name of this kata). Strings passed in will consist of only letters and spaces. Spaces will be included only when more than one word is present.

Solution:

```php
function spinWords(string $str): string {
  $array = explode(" ", $str);
  $result = "";
  for ($i = 0; $i < count($array); $i++) {
    if ($i > 0 && $i != count($array)) {
      $result .= " ";
      }
    if (strlen($array[$i]) > 4) {
      $result = $result . strrev($array[$i]);
    }
    else {
      $result = $result . $array[$i];
    }
  }

  return $result;
}
```

a

Функция в задаче получает строку и должна вернуть эту же строку, но с перевернутыми словами, длина которых больше 4 символов. 

С функцией explode уже был знаком, о существовании strrev догадывался по наитию. Вводную строку разбил на массив слов и последовательно проверял каждое из них на количество символов. Т.к. вводная строка могла содержать одно слово или несколько, пришлось ставить условие, которое ставит пробел, если слов несколько. Условие  ```$i != count($array)``` оказалось избыточным, но это я уже заметил в ходе заполнения сводки.
