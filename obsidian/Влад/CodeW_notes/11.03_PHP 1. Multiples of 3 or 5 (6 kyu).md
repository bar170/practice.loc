If we list all the natural numbers below 10 that are multiples of 3 or 5, we get 3, 5, 6 and 9. The sum of these multiples is 23.

Finish the solution so that it returns the sum of all the multiples of 3 or 5 **below** the number passed in.

Additionally, if the number is negative, return 0.

**Note:** If a number is a multiple of **both** 3 and 5, only count it _once_.

Solution:

```php
function solution(int $number): int {
  $result = 0;
  for ($i = 0; $i < $number; $i += 3) {
    if ($i % 3 != $i % 5){
      $result += $i;
    }
  }
  for ($i = 0; $i < $number; $i += 5) {
    $result += $i;
  }
  return $result;
}
```

Сводка:

Задача на подсчет суммы чисел, которые делятся на 3 или 5 в диапазоне от 0 до числа, передаваемого функции.  Использовались только математические вычисления.

В решениях других использовалось более короткое решение:

```php
function solution($number) {
  $sum = 0;
  for ($i = 3; $i < $number; $i++) {
    if ($i % 3 === 0 || $i % 5 === 0) {
      $sum += $i;
    }
  }
  return $sum;
}
```

Думаю, мое решение немного быстрее работает, т.к. указанный шаг цикла соответствует числам, которые делятся на 3. Можно было ускорить выполнение функции добавив проверку на общий множитель в цикл с делением на 5, т.к. в нем суммарно меньше шагов, чем в цикле с делением на 3.
