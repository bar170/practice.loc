
**Условие задачи**
Нейтан пьет 0.5 литра воды за каждый час езды на велосипеде.
Дано время в часах (может быть дробным), нужно вернуть количество литров, округленное вниз .
Например: 
- 3 часа → 1 литр (3 * 0.5 = 1.5 → округление вниз = 1)
- 6.7 часа → 3 литра (6.7 * 0.5 = 3.35 → округление вниз = 3)

 **Решение задачи**
```php
function litres(float $t): int {
    $total = $t * 0.5;
    $result = floor($total);
    return $result;
}
```

 **Использованные функции**
- [[PHP_floor]] — округляет число вниз
- [[PHP_умножение]] — оператор `*` для умножения
- [[PHP_return]] — возвращает результат
