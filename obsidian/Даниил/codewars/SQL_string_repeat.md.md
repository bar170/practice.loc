
**Условие задачи**
Есть таблица `repeatstr` с двумя колонками:
- `n` — целое неотрицательное число (сколько раз повторить)
- `s` — строка (что повторять)
Нужно написать SQL-запрос, который вернет:
- колонку `s`
- колонку `n`
- колонку `res` — результат повторения строки `s` ровно `n` раз

**Решение задачи**
```sql
SELECT  
    s,  
    n,  
    REPEAT(s, n) AS res  
FROM repeatstr;
```

**Использованные функции/операторы**
- [[SQL_SELECT]] — выбор данных из таблицы
- [[SQL_REPEAT]] — функция повторения строки
- [[SQL_AS]] — создание псевдонима для столбца
