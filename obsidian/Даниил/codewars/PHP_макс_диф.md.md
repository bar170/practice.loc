
**Условие задачи**
Нужно найти разницу между максимальным и минимальным значением в массиве. 
Если массив пустой или содержит только один элемент — вернуть 0.
Массив не отсортирован, могут быть отрицательные числа.

## Решение задачи
```php
function maxDiff(array $list): int {
    $result = 0;
    if (count($list) > 1) {
        $result = max($list) - min($list);
    }
    return $result;
}
```

**Использованные функции**
- [[PHP_max]] — находит максимальное значение в массиве
- [[PHP_min]] — находит минимальное значение в массиве
- [[PHP_count]] — считает количество элементов в массиве
- [[PHP_return]] — возвращает результат
