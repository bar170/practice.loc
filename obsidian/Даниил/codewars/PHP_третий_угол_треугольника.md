
**Условие задачи**
Даны два внутренних угла треугольника. Нужно найти третий. Сумма углов = 180°.

**Решение**
```php
function otherAngle($a, $b) {
    $sumOfAngles = 180;
    $firstAngle = $sumOfAngles - $a;
    $thirdAngle = $firstAngle - $b;
    return $thirdAngle;
}
```
