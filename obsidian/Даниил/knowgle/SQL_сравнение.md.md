
# Операторы сравнения в SQL

Используются для сравнения значений. Возвращают true/false.

Основные операторы:
- `=` равно
- `>` больше
- `<` меньше
- `>=` больше или равно
- `<=` меньше или равно
- `<>` не равно

Пример

```sql
SELECT (mpg * fuel_left) >= distance_to_pump AS res FROM zerofuel;
```
