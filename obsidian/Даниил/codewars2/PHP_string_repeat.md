**Условие задачи**
Написать функцию, которая принимает целое неотрицательное число n и строку s, и возвращает строку, повторённую ровно n раз.

**Решение задачи**

```php
function repeatStr(int $n, string $str): string {
    return str_repeat($str, $n);
}
```
