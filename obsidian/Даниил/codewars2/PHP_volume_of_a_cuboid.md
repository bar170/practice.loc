**Условие задачи**
Бобу нужен быстрый способ посчитать объём прямоугольного параллелепипеда.  
Написать функцию, которая принимает три значения: длину, ширину и высоту, и возвращает объём.

**Решение задачи**

```php
function get_volume_of_cuboid($length, $width, $height) {
$volume = $length * $width * $height;
return $volume;
}
```
