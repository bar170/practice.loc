3)Установка nginx, PHP и их настройка
3.1)Установка nginx и PHP
### Установка nginx и PHP
```bash
# Обновляем список пакетов
sudo apt update

# Устанавливаем nginx
sudo apt install nginx -y

# Проверяем, что nginx запустился
sudo systemctl status nginx
# Должен быть статус: active (running)

# Добавляем в автозагрузку
sudo systemctl enable nginx

# Устанавливаем PHP и менеджер процессов FastCGI
sudo apt install php8.1-fpm php8.1-cli -y

# Проверяем статус php-fpm
sudo systemctl status php8.1-fpm
# Должен быть статус: active (running)

# Добавляем в автозагрузку
sudo systemctl enable php8.1-fpm
```


3.2)Создаем файл конфигурации для нашего домена

sudo nano /etc/nginx/sites-available/daniil333.ru

Далее вставляем туда следующее
server {  
   listen 80;  
   listen [::]:80;  
  
   root /home/admin/sites/daniil333.ru/obsidian/www;  
   index index.php index.html index.htm;  
  
   server_name daniil333.ru www.daniil333.ru;  
  
   location / {  
       try_files $uri $uri/ =404;  
   }  
  
   location ~ \.php$ {  
       include snippets/fastcgi-php.conf;  
       fastcgi_pass unix:/var/run/php/php8.4-fpm.sock;  
   }  
}

3.3)Активация конфигурации

Создаем символическую ссылку в sites-enabled
sudo ln -s /etc/nginx/sites-available/daniil333.ru /etc/nginx/sites-enabled/

Удаляем дефолтный сайт (чтобы не конфликтовал)
sudo rm /etc/nginx/sites-enabled/default

Проверяем правильность конфигурации
sudo nginx -t

Должны увидеть: test is successful

3.4)Проверка конфигурации

Перезагружаем nginx с новыми настройками
sudo systemctl reload nginx

Проверяем, что конфигурация применилась
sudo systemctl status nginx
