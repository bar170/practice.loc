
```php
array_diff($a, $b)
```
Принимает аргументом массив `$a` и массив `$b`. Может принимать аргументами несколько массивов начиная от второго аргумента. Возвращает массив, который содержит только те элементы массива `$a`, которых нет в других указанных массивах начиная с массива `$b`. 
Примечание: ключи элементов нового массива такие же, какие были в массиве `$a`. Т.е. если из массива `$a` был удален первый элемент, то первый элемент нового массива будет начинаться с ключа '1', а не '0'. Соответственно сохраняются и именные ключи по типу 'key_0'.

Пример: 
```php
$result = array_diff([1,2,3,4,5,6], [2,3], [5,6]); 
# $result == [1,4]
```
