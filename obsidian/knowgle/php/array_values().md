
```php
array_values($x)
```
Принимает аргументом массив `$x`. Возвращает этот же массив, но с числовыми ключами элементов в порядке возрастания.

Пример: 
```php
$array = [
	'key_0' => 5;
	'key_1' => 10;
	'key_word' => 4;
]
$array = array_values($array); 
/* $array == [
	'0' => 5;
	'1' => 10;
	'2' => 4;
]
*/
```
