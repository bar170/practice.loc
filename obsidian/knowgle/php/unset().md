
```php
unset()
```
Принимает аргументом любую переменную и стирает содержащееся в ней значение. Может принимать несколько аргументов за раз. При удалении элемента из массива не меняет ключи остальных элементов.

Пример: 
```php
$int = 5;
$string = 'text';
$arr = [22, 44];
unset($int, $string, $arr[0]);
/* $int == null
$string == null
$arr = ['1' => 44]
*/
```
