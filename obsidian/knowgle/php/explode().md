```php
explode($x, $y)
```
Принимает аргументами строку ```$x``` и строку `$y`. Удаляет из строки `$y` значения строки `$x`. Возвращает массив со строками, на которые разбила строку `$y`.

Пример: 
```php
$arr = []; #Пустой массив
$arr = explode(' ','Каждое слово запишется отдельным элементом массива');
/*
$arr == 
[
	'Каждое', 
	'слово', 
	'запишется', 
	'отдельным', 
	'элементом', 
	'массива'
];
*/
```
