# INSERT

Оператор SQL для добавления новых записей в таблицу.

**Принимает** аргументом название таблицы, список полей и список значений.

**Возвращает** количество добавленных строк.

Оператор может принимать дополнительные аргументы:
- `DEFAULT VALUES` — вставить строку со значениями по умолчанию
- `ON CONFLICT` — поведение при конфликте
- `RETURNING` — вернуть вставленные данные

**Пример**

```sql
INSERT INTO cars (name, brand, price) VALUES ('Model S', 'Tesla', 80000000);
```
добавит одну строку
