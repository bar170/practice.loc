```git
git branch
```
Показывает все локальные ветки. Текущая ветка будет помечена звездочкой (\*)
Флаги:
-a покажет локальные и удаленные ветки
\--merged покажет ветки, которые уже объединены с текущей
\--no-merged покажет ветки, которые еще не объединены с текущей
-m \[oldName] \[newName] переименовывает текущую ветку
-d \[branch] удаляет ветку, если она уже была объединена с текущей веткой
-D \[branch] принудительное удаление ветки (в случае, если коммиты уже созданы, но изменения еще не отправлены)
-f (или --force) сбрасывает ветку до стартовой точки (сбрасывает изменения)

```git
git branch [newBranch]
```
Создает новую ветку \[newBranch], но не переключается на нее
