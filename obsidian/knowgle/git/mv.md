```git
git mv [path/file] [newpath/]
```
Перемещает и переименовывает файл, автоматически обновляет индекс
Некоторые флаги:
-f (или --force) перезаписывает целевой файл, если тот уже существует
-n (или --dry-run) показывает результат команды без ее применения 
Пример: 
```git
git mv /home/file.txt /home/admin/movedFile.txt
```
