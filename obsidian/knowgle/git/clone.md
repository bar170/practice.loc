```git
git clone [flags] <url> [catalog]
```
Клонирование репозитория в новый каталог.
\[catalog] - имя целевой папки (опционально)
\<url> - адрес удаленного репозитория
\[flags] - флаги (указываются через знак -)
 Пример: 
 ```git
 git clone https://github.com/bar170/practice.loc.git home/admin/obsidian
 ```
